## .$get()函数

jQuery 中的 .$get() 函数专门用来发起 get 请求，从而将服务器上的资源请求到客户端来进行使用

```javascript
$.get(url,[data],[callback])
```

| 参数名   | 参数类型 | 是否必选 | 说明 |
| -------- | -------- | -------- | ---- |
| url      | string   | 是       |      |
| data     | object   | 否       |      |
| callback | function | 否       |      |
